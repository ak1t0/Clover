# Clover

Clover is a Clojure-like syntax compiler.

## Installation

Clover requires Go(>= 1.4) and GHC(>= 7.10) and Cabal(>= 1.22).

    git clone https://github.com/ak1t0/Clover
    cabal build

## Usage


    dist/build/clover/clover -s source output
    dist/build/clover/clover --help
